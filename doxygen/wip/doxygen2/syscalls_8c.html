int abi_rule_add ( struct db_filter *  db,
struct db_api_rule_list *  rule 
)

add a new rule to the abi seccomp filter

Parameters
dbthe seccomp filter db
rulethe filter rule

This function adds a new syscall filter to the seccomp filter db, making any necessary adjustments for the abi ABI. Returns zero on success, negative values on failure.

It is important to note that in the case of failure the db may be corrupted, the caller must use the transaction mechanism if the db integrity is important.
