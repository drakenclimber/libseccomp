Resolve a syscall name to a number

Parameters
archthe architecture definition
namethe syscall name

Resolve the given syscall name to the syscall number using the syscall table. Returns the syscall number on success, including negative pseudo syscall numbers; returns __NR_SCMP_ERROR on failure.
